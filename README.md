(Updated) APK yalnızca senin kullanımın için: ekledim sunucu tarafı token kontrolü ve mobil config örneği. - Sunucu: ayarlamak için: export API_TOKEN=your_secret_token_here
- Mobil (APK) için: static/config.json içine aynı tokenı koy veya CI ile inject et. - Güvenlik: token APK içinde bulunur; APK yalnızca senin dağıtımın için kullanılmalı.
